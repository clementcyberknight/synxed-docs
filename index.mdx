Synxed gives developers a simple SDK to embed licensed music playback directly into their applications. Whether you need a drop-in floating player or a fully custom UI built from scratch, Synxed handles licensing, streaming, ads, voice commands, and resumable playback — so you can focus on building.

## How It Works

Synxed authenticates your app with an API key from the [Synxed Portal](https://portal.synxed.com). You choose a source — a song ID, playlist code, mix ID, live radio, or station — and the SDK handles everything else.

<Steps>
  <Step title="Get your API key">
    Sign up at [portal.synxed.com](https://portal.synxed.com) and copy your API key.
  </Step>
  <Step title="Install the SDK">
    ```bash
    npm install synxed-sdk
    ```
  </Step>
  <Step title="Mount the player">
    ```ts
    import "synxed-sdk/player.css";
    import { SynxedWebPlayer } from "synxed-sdk";

    SynxedWebPlayer.mount({
      apiKey: "YOUR_SYNXED_API_KEY",
      source: { type: "playlist", playlistCode: "sxpl_YOUR_CODE" },
    });
    ```
  </Step>
  <Step title="Ship it">
    Call `player.destroy()` on unmount and you're done. See the [production checklist](/production-checklist) for best practices.
  </Step>
</Steps>
